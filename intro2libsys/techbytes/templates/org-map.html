Pat Wagner, of Pattern Research Inc., provides a useful management model for library organizations. task, managerial, and leadership levels in a library demand similar and different requirements from library systems and and supporting technologies. Time

Task Level

  • Professional

    Reference Desk A reference librarian assisting a patron with a question using the library's catalog

    Technical Services An acquisition staff members orders a book request of a patron and creates a stub record in the library's ILS

  • Technical

    Library ILS Support makes sure the library's online catalog and ILS is properly backed-up.

Managerial Level

Reference Department Head
  • Communicate results of Reference Desk reference interviews for the past year using the library's catalog to the System Department Head.
  • Coordinate Reference Desk schedule for the next month.
  • Translate actions and outcomes of the Reference Desk operations to library leadership and other departments like Technical Department Head and staff in Library ILS Support

Leadership Level

Library Director
  • Vision
  • Risk
  • Influence
