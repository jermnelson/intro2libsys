A through M

Aristotle Library Apps
An open-source project that provides an HTML5 front-end app environment for the MARCR Datastore project. Both projects were started and are actively being development by Colorado College.
Bibliographic Framework Initiative Approach for MARC Data as Linked Data
These are the slide notes from a presentation by Sally McCallum from the Library of Congress.
Bootstrap
A web front-end framework used to provide responsive HTML5 apps in the Aristotle Library Apps project.
DevOps: What It Is, Why It Exists and Why It's Indispensable
A blog posting on ReadWriteWeb by Luke Kanies that explains the DevOps movement in IT management for enterprise information technology.
Dive into HTML5
An ebook original written by Mark Pilgrim and now kept updated by an ad hoc group of maintainers is a good introduction to many of the new features that HTML5 brings to web development.
Django
A high-level Python Web framework used in the Aristotle Library Apps project.
The Manifesto for Agile Software Development
A listing of AGILE software development core principles.
MARCR Datastore
The link to the Github project for MARCR Datastore, a Redis-based opens source project for managing bibliographic records and operational data.

N through Z

Pattern Research Inc,
This is the website for Pat Wagner's company, Pattern Research, and includes many links and resources related to her work and model of an organizational map.
Redis
Website for the NoSQL Redis project. Includes downloads, documentation, and tutorials for using Redis.
What the f**k is lean startup and why should I care?
These are the slide notes from a presentation by Laurence McCahill of Spook Studio.
