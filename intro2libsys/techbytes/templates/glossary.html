A through M

AACR2
An acronym for Anglo-American Cataloging Rules the current bibliographic cataloging practices used by most libraries in some form or another for cataloging material into the MARC21 bibliographic data format.
ALA
The American Library Association is the largest library professional organization that is actively involved in most areas of library operations.
Django
Django is an a Python-based web framework that is used in the Aristotle Library Apps project.
FRBR
Functional Requirements for Bibliographic Records is an Entity-Relationship model for bibliographic records that was first released in 1998 by an IFLA study group. FRBR is composed of four main entities; Work, Expression, Manifestation, and Item also known as WEMI along with various other entities to support creator, publisher, and subject authority relationships with the core WEMI entities.
HTML5

IFLA
An acronym for International Federation of Library Associations and Institutions an international professional organization.
Linked Data
Linked Data is an international effort by multiple user communities to build a semantic web for representing relationships between entities and objects on the internet.
MARC21
An acronym for Machine Readable Cataloging record, MARC21 is the most widely used data-carrier format and specification used by libraries today.
