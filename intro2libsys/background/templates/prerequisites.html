    Web technologies

    Before the first day of Introduction to Library Systems, you should be familiar with HTML 5, Javascript, and XML.

    We will be using Django and Python programming language for class assignments and projects. Download and install both Python and Django, the tutorial in the basic Python document provides an overview of Python.

    {% endblock %} {% block one-two-box %}

    Cloud Web Server & Git

    You are expected to have a virtual Linux server for your assignments and projects. Rackspace and Amazon Web Services EC2 are good options.

    You will be creating a guest user on your Linux server (jpnelson is a good user name) so I can run and test your code and assignments in your environment.

    All of your creative work for this course will be managed with a distributed version control system called Git.
