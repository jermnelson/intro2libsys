Catalog Pull Platform's

Semantic Server

A critical component of the Catalog Pull Platform is the Linked-Data and JSON-based Semantic Server. The Semantic Server provides a REST API for managing different artifacts vocabularies that are stored internally in JSON-base documents. The core code-base of the Semantic Server is licensed under the MIT software license with other components licensed under various open-source or open-source friendly licenses.
