Current Projects

Collaborating with Aaron Schmidt of Influx Library User Experience on BIBCAT, a BIBFRAME catalog for the Library of Congress. The current emphasis is building a Semantic Server that provides an common REST API to entities structured as BIBFRAME, Schema.org, and MARC JSON-LD.

As the lead architect and software engineer for individual components of the CPP, I'm working towards a soft launch of the Colorado College TIGER Catalog with a first MVP release now available at catalog.coloradocollege.edu.
