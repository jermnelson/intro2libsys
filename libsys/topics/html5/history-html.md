The History of HTML
-------------------
[HTML][HTML], an acronomyn of Hyper-text Markup Language, started as an experiment by Sir Tim Berners-Lee
in 1989 at CERL. 


## Basic HTML Syntax
<code>
<html>
  <head></head>
  <body></body>
</html>
</code>
